windows常用命令

一些有用的命令

文件操作

  • # 将ipconfig的执行结果输出到C:/1.txt
  • ipconfig >> C:/1.txt
  • # 查看文件
  • type C:/1.txt
  • # 全盘搜索文件(盘符自己把握)
  • dir c:\ d:\ e:\ /s /b | find "简历.pdf"

系统用户相关

  • # 查询当前服务器所有的账户
  • net user
  • # 查看计算机用户组列表
  • net localgroup
  • # 查询指定用户的具体信息
  • net user username
  • # 创建一个新的账户
  • net user username password /add
  • # administrators(想要提升到的用户组) cs(想要提升到指定用户组的账户)
  • net localgroup administrators cs /add
  • # 删除指定用户
  • net user user /del

信息查看相关

  • # 查看系统版本信息
  • systeminfo
  • # 查看系统开放端口
  • netstat -ano
  • # 查看80端口
  • netstat -aon|findstr "80"
  • # 查看2448进程
  • tasklist |findstr "2448"
  • # 查看系统进程
  • tasklist /svc
  • ipconfig /all
  • whoami

进程端口相关

  • # 列出当前计算机运行的程序
  • tasklist
  • taskkill /svc
  • # 结束指定程序
  • taskkill /f /im xx.exe
  • # 结束pid为111的端口
  • taskkill /f /pid 111

查看历史打开文件

  • dir %APPDATA%\Microsoft\Windows\Recent

其他命令

  • 查看版本
  • ver
  • 查看主机名
  • hostname
  • 清屏
  • cls
  • 查看服务
  • net start
  • 关闭防火墙
  • net stop sharedaccess
  • 查看当前机器的环境变量
  • set
  • 列出当前机器上的所有盘符
  • fsutil fsinfo drives
  • 搜集注册表中的各种密码数据
  • reg query HKLM /f password /t REG_SZ /s
